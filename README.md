# Meu Portfolio

Projeto de portfólio pessoal para apresentar minha experiência, projetos, habilidades e informações de contato.

### 🌐 [Disponível aqui!](https://gustavoolsantos.vercel.app)

## Tecnologias utilizadas

- React
- HTML
- CSS
- JavaScript
- Cloudnary para hospedar o currículo

## Como executar

1. Clone este repositório:

   ```bash
   git clone https://github.com/seu-usuario/gofer-portfolio.git
   ```
2. Execute o comando `npm run dev`
